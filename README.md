### 🚀 High-Impact Projects

#### 🛡️ [Sentinel: Self-Healing Knowledge Graph](https://github.com/Om7035/Sentinel-The-Self-Healing-Knowledge-Graph)
*AI Infrastructure | Python • Neo4j • LangChain*
- **The Challenge:** Traditional RAG models rely on static vector embeddings that become obsolete as soon as the source data changes.
- **The Solution:** Built a "Self-Healing" pipeline that uses change-data-capture (CDC) to trigger LLM-based re-indexing of specific graph nodes, maintaining 99% data freshness without full re-embeds.
